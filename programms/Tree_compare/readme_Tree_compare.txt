This repository contains scripts to compare with a compare result as a tree

1. compare_documentation_tree_text_files.lua

This graphical user interface is useful for the comparison of two text files or two texts in text boxes. The result is a tree. Trees in text files can be compared with this.

2. compare_documentation_tree_lua_trees.lua

This graphical user interface is useful for the comparison of two trees. The result is also a tree.

3. Git_download_files_to_compare_IDIV.lua

This script downloads all the files on github documentation tree other versions and compares them with local files.
