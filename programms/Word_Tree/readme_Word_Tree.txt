This repository contains scripts with a tree in Word

1. Tree_word_build_luacom.lua

This script builds from a Lua table a tree in Word with LuaCOM.

2. Tree_word_build_luacom_local_DE.lua

This script builds from a Lua table a tree in Word with LuaCOM. The script is written for german windows version.

3. Word_VBA.txt

This file contains VBA-code to copy in Word docm for the interactive-dynamic table of contents in do it yourself manner.

4. Word_Tree.docm

This file contains a tree in VBA Word.
