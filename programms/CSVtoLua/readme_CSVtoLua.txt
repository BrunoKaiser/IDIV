This repository contains Lua scripts to build a Lua table from a csv file with dependencies between child and parent node

1. example_dependemcies.csv

The csv files must habe two columns: 1. child and 2. parent. There must be only one root for the whole tree build by dependencies.

2. CSVtoLua_Linux.lua

Script for Linux system.

3. CSVtoLua_Windows.lua

Script forWindows system.
